WebSite "Autoblog":

Projeto final desenvolvido para a disciplina de Programação para internet.

É um website funcional onde o usuário pode ver notícias automotivas e imagens, com sistema de duvidas, login funcionais e barra de pesquisa.

FUNCIONALIDADES:

Barra de pesquisa funcional, sistema onde o usuario pode enviar duvidas e um lugar onde elas podem ser visualizadas, e um sistema de login funcional, onde o administrador pode inserir imagens na galeria.

TECNOLOGIAS UTILIZADAS:

HTML, CSS, PHP (Com CRUD) e MySQL com banco de dados funcional (Via XAMPP PhpMyAdmin)

OBJETIVO DO PROJETO:

Praticar design de aplicações web, praticar e fixar o conceito de CRUD e o uso de banco de dados

COMO EXECUTAR:

(Existe um LeiaMe detalhado dentro do arquivo)

Abrir o projeto no VSCode, e executar usando uma extensão chamada OpenPHP e o XAMPP
